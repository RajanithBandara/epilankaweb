# 🦠 EpiWatch Lanka
### Infectious Disease Awareness & Prediction Platform
A mobile-friendly web platform for visualizing, analyzing, and predicting infectious disease spread across Sri Lanka using historical epidemiological data.

---

## 📖 Overview

EpiWatch Lanka is a modern, intelligent disease surveillance platform designed to help the public, travelers, and health authorities monitor real-time disease trends in Sri Lanka.  
The system provides:

- District and province–level disease visualizations
- Interactive heatmaps
- Historical trend analysis
- AI-based prediction of future outbreaks
- Public awareness & health guidelines
- Alerts for high-risk regions

The platform is built using **Next.js (Frontend)** and **FastAPI (Backend)** with an extendable **Machine Learning layer** for forecasting future disease spread.

---

## 🎯 Project Objectives

- Collect & analyze historical disease case data (Dysentery, Meningitis, Tuberculosis, etc.)
- Build an interactive dashboard for disease visualization
- Implement a prediction engine using time-series forecasting
- Provide real-time alerts & preventive guidelines
- Support mobile-first user experience

---

## 🏗️ Tech Stack

### **Frontend**
- **Next.js** (React framework)
- **Tailwind CSS** (styling)
- **Leaflet.js** (interactive maps)
- **Chart.js / Recharts** (graphs and charts)

### **Backend**
- **FastAPI** (Python API server)
- **PostgreSQL** (disease, climate & demographic datasets)
- **MongoDB** (user accounts, preferences, and notifications)

### **Machine Learning**
- Time-series forecasting (SARIMA, Random Forest, or LSTM)
- Outbreak risk scoring
- Pattern and seasonality detection

### **Deployment**
- Vercel / Netlify (Frontend)
- Railway / Render / Azure (Backend)
- MongoDB Atlas / Railway PostgreSQL (Databases)

## 🎨 UI/UX Design Theme

The application follows a **Wine Red + Black + White** color scheme:

- **Primary:** Wine Red (`#A41111`)
- **Dark:** Deep Red (`#8B0000`)
- **Secondary:** Black / Charcoal (`#0D0D0D`, `#1A1A1A`)
- **Background:** White & Light Grey
- **Accent:** Light Red & Salmon shades

Designed for:  
✔ Mobile-first experience  
✔ Clean data visualization  
✔ High readability with strong contrast
